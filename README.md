# Lexington Chinese School — First Demo

A responsive React + Vite + TypeScript first-demo website based on recoverable content from the existing Lexington Chinese School website and the information architecture of the Fayette County Public Schools homepage.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important content warning

The old school domain was unavailable during extraction. Search-engine-indexed pages and archived registration materials supplied the initial content. Before publishing, school leadership must verify:

- Current address and classroom venue
- Meeting day and hours
- Phone and email
- Tuition and fees
- Current class levels and enrichment offerings
- Staff and board information
- Academic calendar and registration dates
- All downloadable forms

Editable content is centralized in `src/data/siteData.ts`.

## Design direction

- FCPS-inspired clear navigation, prominent shortcuts, announcements, news/calendar, and substantial footer
- Smaller, warmer community-school identity
- Deep red, charcoal, cream, and restrained gold
- No database or authentication yet
